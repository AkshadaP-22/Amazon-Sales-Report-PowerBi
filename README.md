# Amazon Sales Report - Power BI Dashboard

## Project Overview
This project uses an E-Commerce sales dataset to analyze online business performance through sales trends, category insights, and shipping analysis using Power BI.

## Key Features
- KPI Cards (Sales, Quantity, Orders)
- Category-wise Sales Analysis
- State-wise Performance
- Courier/Shipping Insights
- Interactive Slicers

## Tools Used
- Power BI Desktop
- Excel Dataset
